# Drip VRC — Site

Minimal, modern black & white landing page.

How to use
- Open `index.html` in a browser to preview.
- Replace the placeholders in `index.html`:
  - `#REPLACE_WITH_DISCORD_LINK` — your Discord invite or server URL
  - `#REPLACE_WITH_VRCHAT_LINK` — your VRChat group URL or world link

Files added
- `index.html` — landing page with embedded SVG logo and icons
- `styles.css` — black & white theme and layout
